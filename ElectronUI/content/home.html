Welcome to SnitchIO

This program installs a service on Windows machines to monitor processes and applications to alert system administrators via email of potential threat actor activity. It must be ran as administrator to install or uninstall the service.

Monitoring and Alerting Functionality

  • Powershell
  • Command prompt (CMD.exe)
  • Remote Desktop Protocol (RDP)
  • Application Installation

Prerequisites

  • Windows Server 2012 R2 and greater
  • windows 8.1/RT 8.1 and greater
